## 9. TCP/IP

–ö–æ–≥–∞—Ç–æ –≥–æ–≤–æ—Ä–∏–º –∑–∞ HTTP, –Ω–µ –º–æ–∂–µ–º –¥–∞ –ø—Ä–æ–ø—É—Å–Ω–µ–º –æ—Å–Ω–æ–≤–∞—Ç–∞, –≤—ä—Ä—Ö—É –∫–æ—è—Ç–æ —Ç–æ–π —Å—Ç—ä–ø–≤–∞ ‚Äì
